Sobre o projeto

Este projeto tem como objetivo criar um jogo da memória digital para ajudar crianças dos primeiros anos do Ensino Fundamental a melhorarem suas habilidades de leitura. A ideia é usar a tecnologia de forma divertida e interativa, aproveitando os benefícios das Tecnologias Digitais de Informação e Comunicação (TDICs) na educação.

Durante a pandemia de COVID-19, muitos alunos ficaram sem acesso à escola, o que prejudicou ainda mais o aprendizado de leitura, especialmente entre as crianças menores. Este jogo foi pensado para ajudar a recuperar essa defasagem, usando atividades lúdicas para tornar o processo de alfabetização mais atraente.

O jogo funciona assim: a criança associa palavras às figuras e pode ouvir a pronúncia correta, o que facilita o aprendizado. Todo o desenvolvimento do projeto seguiu métodos ágeis (Scrum e Kanban) para garantir que o jogo fosse produzido de forma eficiente e organizada. Ele foi desenvolvido com HTML, CSS e Bootstrap no visual, enquanto JavaScript e Java foram usados para programar o funcionamento do jogo e o banco de dados.

Professores, especialmente da rede pública, testaram o jogo e deram feedback positivo, afirmando que ele tem grande potencial para ajudar no aprendizado de leitura. O próximo passo será melhorar o jogo com novas funções e testá-lo em diferentes ambientes escolares para expandir seu uso.

Sabemos que é um projeto simples e que não acabará de vez com toda a problemática. Mas acreditamos que um simples gesto pode fazer a diferença!
